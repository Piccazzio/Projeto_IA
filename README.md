### Descrição do Projeto

Este projeto tem como objetivo o desenvolvimento de um modelo computacional para detecção de discurso de ódio em redes sociais, utilizando técnicas de Processamento de Linguagem Natural (PLN) e, futuramente, Redes Neurais em Grafos (GNN).

Nesta primeira etapa, o foco está na definição do problema como uma tarefa de classificação binária (ódio vs. não ódio), bem como na preparação e análise exploratória dos dados. Foram realizadas atividades como limpeza textual, normalização dos dados, rotulagem manual e análise estatística inicial, visando compreender melhor o comportamento da base de dados.

### Dataset

O projeto utiliza um conjunto de dados contendo publicações de redes sociais. Inicialmente, os dados não possuíam rótulos para identificação de discurso de ódio, sendo necessária a realização de um processo de rotulagem manual pelos integrantes do grupo, classificando cada instância como “ódio” ou “não ódio”.

Importante: Devido à natureza sensível dos dados utilizados, que contêm informações confidenciais, o dataset não está disponível neste repositório público. A disponibilização desses dados é restrita para fins acadêmicos e segue diretrizes éticas e de privacidade.
